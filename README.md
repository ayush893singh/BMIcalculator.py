HealthCheck BMI Calculator

A simple Python project that calculates Body Mass Index (BMI) using a user's weight and height.

Features

- Calculate BMI instantly
- Height input in centimeters (cm)
- Weight input in kilograms (kg)
- Displays BMI category
- Beginner-friendly Python project

This project uses:

- Output Formatting
- Basic Python Syntax

---

Formula Used

BMI is calculated using the following formula:

BMI = Weight (kg) / Height² (m²)

Example:

Weight = 63 kg

Height = 180 cm = 1.80 m

BMI = 63 / (1.80 × 1.80)

BMI = 19.44

How It Works

1. User enters weight in kilograms.
2. User enters height in centimeters.
3. Height is converted from centimeters to meters.
4. BMI is calculated.
5. Program displays BMI value and health category.

18.5 – 24.9| Normal Weight
25 – 29.9| Overweight
30 and above| Obese
